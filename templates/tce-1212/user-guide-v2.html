User guide

1. Overview

The dm+d is a dictionary of descriptions and codes which represent medicines and devices in use across the NHS. The dm+d data is refreshed weekly and contains a huge variety of information, including:

  1. whether a product will be reimbursed by NHSBSA, if submitted for reimbursement by a dispensing contractor
  2. the indicative price of each pack of a product (where a price is maintained)
  3. current and discontinued products and packs available from manufacturers and Suppliers

The user has the following options:

  1. Product Search
  2. Ingredient Search
  3. Code Search

2. General Information

Throughout the browser the user can refine searches using a mixture of both Drop down boxes and radio buttons.

The criteria entered by the user does not have to conform to regular dm+d naming convention of name/strength/form e.g., “Paracetamol 500mg tablets”. The system can return results where partial information is entered e.g., “Paracetamol tablets 500mg” or “Para 500 tab”.

There are ‘breadcrumbs’ throughout the site which allows the user to quickly navigate to previous screens using the hyperlinks an example is shown below.

In some places a Copy icon can be seen which allows the user to quickly copy the entry. Below shows the SNOMED code with the copy icon at the end of the code.

Search results default 20 entries but the user has the option to change to view 50 or 100 results.

The data displayed throughout the dm+d browser reflects the data held by the NHSBSA.

By default, Invalid products are not shown – to show Invalid products tick the box labelled ‘Show Invalid’.

Discontinued AMP, Parallel Imports and Special Orders are shown by default – to hide these in the search results, tick the relevant box before clicking search.

Historical information is displayed where the data is held by the NHSBSA.

3. Product Search

The product search can be used to search for

  1. VTM
  2. VMP
  3. AMP

3.1 VTM

Once the drop down is set to VTM the user can enter the product name

Inserting the product name and clicking search button returns a list of all products matching the search criteria.

By default, Invalid products are not shown, however, users can choose to display Invalid products by ticking the box labelled ‘Show Invalid’.

The result table provides the VTM Name and SNOMED Code of products which match the search criteria entered.

The user can navigate to any of the products listed by clicking the relevant VTM Name. The results screen below will be displayed to the user.

From this screen the user can navigate, using the hyperlinks, to any of the VMP products listed or see further information by selecting the further links hyperlink.
