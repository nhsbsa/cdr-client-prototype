View wild card Information

These are the most commonly used wildcard symbols used on the system

  • % (Percent Sign) – Matches any number of characters. Example: %cet finds "paracetamol", "acetaminophen".
  • , (Comma) – Separates multiple search criteria. Example: paracetamol,acetaminophen finds either "paracetamol" or "acetaminophen"
  • _ (Underscore) – Matches exactly one character. Example: p_raetamol finds "paracetamol" but not "ptraetamol".
  • [] (Square Brackets) – Matches any one character in a set or range. Example: par[ae]tamol finds "paracetamol" or "paratamol".
  • {} (Curly Braces) – Matches a specific number of repetitions. Example: a{2} finds "paracetamol" but not "paracetalol".
  • - ^ (Caret) – Matches the beginning of a string. Example: ^paracetamol finds "paracetamol" at the start of a string..
  • - (Hyphen) – Indicates a range or excludes terms. Example: paracetamol -acetaminophen finds "paracetamol" but excludes "acetaminophen"
If you need to use a wildcard, include the symbol where needed: before, after, or on both sides of your query (for example, %a, a%, or %a%). Otherwise, simply enter the criteria you want to search for.
